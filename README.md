# art-gallery-web-platform using mongodb
Art gallery website using PHP and MongoDB

# Art Gallery Web Platform

## Tech Stack
- PHP
- MongoDB
- HTML, CSS

## Features
- Artists can list and manage their artwork
- Dynamic data handling using backend
- Explored both SQL and NoSQL databases

## How to Run
1. Install XAMPP or any local server
2. Place project in htdocs folder
3. Import database (if applicable)
4. Run on localhost

## Note
This project was built to understand backend development and database management.
